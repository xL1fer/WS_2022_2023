Animal Scouter

Developed in the context of the first project of Semantic Web (WS)
Engineering Masters 2023

University of Aveiro

Configs

Provides users with the ability to manage and update the database. Allows users to add or remove data to and from the database. Users can enter new information into the database, such as new animals or new categories and also delete data deemed outdated or inaccurate.

Any new species?

Queries

Presents drop-down menus, users can select a particular group of animals they are interested in, such as animal class, type of nurturing or number of legs. Users can then click on the name of an individual animal to access all the information stored about that animal.

Try it out!

Ask

A simple and intuitive way to inquire about the presence of specific features or attributes in different animals within the database. By selecting an attribute such as "venomous" and typing in the name of the animal, users can submit a query and receive information about whether or not it is true.

Try it here!

Inferences

Contains a page with a structured table of classes to demonstrate the power of inferences. Also has the functionality of getting extra information from DBpedia and Wikidata in order to maximize the amount of information presented to the user, giving external resources if deeper knowledge is desired.

Check it here!
